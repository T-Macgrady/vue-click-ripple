# vue-click-ripple
vue点击波纹特效插件


## 安装

#### npm下载
`````
● yarn add vue-click-ripple
`````
### vue-cli内main文件

``````
● import Vue from 'vue'

● import clickRipple from "vue-click-ripple"

● Vue.use(clickRipple)
````````````````

## 使用

* this.$clickRipple(ele)  ||  Vue.$clickRipple(ele)

* 参数ele为字符串格式  可以是tag/class/ID name
